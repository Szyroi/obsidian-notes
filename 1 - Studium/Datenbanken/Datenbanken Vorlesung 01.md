### FlippedClassroom
#### Definitionen
Miniwelt = Ausschnitt einer (fiktiven/tatsächlichen) Wirklichkeit
Modell = Werkzeugkasten zur Beschreibung der Miniwelt
Schema = Darstellung mittels Modell (z.B. Diagrammsprache)
System = Verwirklichung von Schema

Modell-Validadation = Prüfen ob Modell eine sinnvolle Repräsentation (= Schema) der zu modellierenden Miniwelt liefert
Modell-Verifikation: Prüfen, ob Schema sich wie gewünscht verhält

Schema ist Korrekt wenn alles Definierte stimmt
Schema ist Vollständig wenn alles Relevante definiert wurde

Vorbild = Vorschreibendes Schema
Abbild = Beschreibendes Schema
