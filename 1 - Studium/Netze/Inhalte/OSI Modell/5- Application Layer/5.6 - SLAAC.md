Lokale Adressbildung Der Host erzeugt eine Link-Local-Adresse der Form FE80::<Interface-ID> und überprüft sie mit der Duplicate Address Detection (DAD) (Abschnitt 5.4 in [24]). 2. Erkennung erreichbarer Router Router senden regelmäßig Router Advertisements (RA) an FF02::1 („All Nodes“). Der Host kann zusätzlich eine Router Solicitation (RS) an FF02::2 („All Routers“) senden, um die Antwort zu beschleunigen. Die Quelladresse des empfangenen RAs wird als Standardrouter gespeichert. 3. Empfang von Präfixinformationen Über Optionen im RA, insbesondere die Prefix Information Option (PIO), erhält der Host Netzpräfixe und Flags.
