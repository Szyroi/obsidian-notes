## Carrier Sense Multiple Access (CSMA)
-  Mach ein "Listen before Talk"
- Es besteht weiterhin die Möglichkeit, dass ein Node, das Medium als frei ansieht, obwohl ein anderer Node sendet. 
- Dies liegt an der **Ausbreitungsverzögerung** (AV). 
- Beide Nodes senden und es kommt zu einer Kollision (Überlagerung von Pegeln, Bitfehler).
- Auf einem Kabel können solche Kollisionen frühzeitig erkannt werden — Collsion Detection.
## Carrier Sense Multiple Access/Collision Detection (CSMA/CD)

- Sobald, eine Kollision vom Sender erkannt wird, bricht dieser die Übertragung des Rahmens ab. 
- Der Sender beginnt damit, ein “JAM”-Signal zu senden. 
- Dieses “JAM”-Signal dient dazu alle Nodes (insb. alle Sender) über die Kollision zu informieren. 
- Rahmen können so als kollidiert gekennzeichnet werden und von den Sendern im Anschluss an das JAM”-Signal wiederholt werden.
