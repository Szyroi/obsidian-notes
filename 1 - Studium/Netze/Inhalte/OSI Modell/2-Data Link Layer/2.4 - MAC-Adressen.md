## Struktur
MAC = Medium Access Controll
48 Bit (6 Byte)
- 24 bit Hersteller-ID (links)
	- 6 bit ID
	- Flag: Is Software-configured address
	- Flag: Is group/broadcast
	- 16 bit ID
- 24 bit Station-ID (rechts)

| U/L         |    2.    |          |
| OUI         | 3. -24.  |          |
| OUA         | 25. -48. |          |
