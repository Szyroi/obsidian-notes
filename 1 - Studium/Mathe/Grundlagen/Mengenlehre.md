# Mengenlehre

## Definition
Eine **Menge** ist eine **Zusammenfassung von unterscheidbaren Objekten**, die als Elemente der Menge bezeichnet werden.  
- Notation: `$A = \{1, 2, 3\}$`  
- Ein Objekt `$x$` ist **Element** einer Menge `$A$`, geschrieben `$x \in A$`.  
- Wenn `$x$` **kein Element** von `$A$` ist: `$x \notin A$`.

## Arten von Mengen
- **Endliche Menge**: hat eine abzählbare Anzahl von Elementen, z.B. `$A = \{1,2,3\}$`  
- **Unendliche Menge**: z.B. Menge der natürlichen Zahlen `$ \mathbb{N} = \{1,2,3,\dots\} $`  
- **Leere Menge**: Menge ohne Elemente, geschrieben `$ \emptyset $` oder `$ \{\} $`  
- **Teilmenge**: `$ A \subseteq B $` bedeutet, jedes Element von `$A$` ist auch in `$B$`  
- **Echte Teilmenge**: `$ A \subset B $`, `$ A \neq B $`

## Mengenoperationen
- **Vereinigung**:  
$$
A \cup B = \{ x \mid x \in A \text{ oder } x \in B \}
$$  

- **Schnitt**:  
$$
A \cap B = \{ x \mid x \in A \text{ und } x \in B \}
$$  

- **Differenz**:  
$$
A \setminus B = \{ x \mid x \in A \text{ und } x \notin B \}
$$  

- **Komplement**:  
$$
A^c = \{ x \mid x \notin A \} \quad (\text{bezogen auf Universum } U)
$$  

- **Kartesisches Produkt**:  
$$
A \times B = \{ (a,b) \mid a \in A, b \in B \}
$$

## Gesetze der Mengenlehre
- **Kommutativgesetze**  
$$
A \cup B = B \cup A, \quad A \cap B = B \cap A
$$  

- **Assoziativgesetze**  
$$
(A \cup B) \cup C = A \cup (B \cup C), \quad (A \cap B) \cap C = A \cap (B \cap C)
$$  

- **Distributivgesetze**  
$$
A \cap (B \cup C) = (A \cap B) \cup (A \cap C), \quad A \cup (B \cap C) = (A \cup B) \cap (A \cup C)
$$  

- **De Morgansche Gesetze**  
$$
(A \cup B)^c = A^c \cap B^c, \quad (A \cap B)^c = A^c \cup B^c
$$

## Spezielle Mengen
- **Natürliche Zahlen**: `$ \mathbb{N} = \{1, 2, 3, \dots\} $`  
- **Ganze Zahlen**: `$ \mathbb{Z} = \{\dots, -2, -1, 0, 1, 2, \dots\} $`  
- **Rationale Zahlen**:  
$$
\mathbb{Q} = \left\{ \frac{p}{q} \mid p \in \mathbb{Z}, q \in \mathbb{N} \right\} 
$$  

- **Reelle Zahlen**: `$ \mathbb{R} $`  
- **Komplexe Zahlen**: `$ \mathbb{C} $`
