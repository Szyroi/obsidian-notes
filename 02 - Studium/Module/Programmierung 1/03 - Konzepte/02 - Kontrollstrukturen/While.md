# While Schleifen

## Erklärung
- [[06_Kontrollstrukturen.pdf]]
- `while()` Wiederholt Code, solange Bedingung true ist
-  Bedingung wird **vor** jedem Durchlauf geprüft

## Syntax

```Java
while (bedingung) {
    // Codeblock
}
```
