# Switch (Java)

## Erklärung
- Alternative zu mehreren `if-else if`
- Prüft Gleichheit von Variablenwerten
- `default` = Standardblock, wenn kein Fall zutrifft

## Syntax
```java
switch (variable) {
    case wert1:
        // Code
        break;
    case wert2:
        // Code
        break;
    default:
        // Code
}
