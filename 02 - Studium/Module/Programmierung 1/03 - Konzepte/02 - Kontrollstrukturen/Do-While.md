# Do-While (Java)

## Erklärung
- Schleife prüft Bedingung **nach** Codeausführung  
- Mindestens 1 Durchlauf garantiert
- Vorlesung: [[06_Kontrollstrukturen.pdf]]

## Syntax
```java
do {
    // Codeblock
} while (bedingung);
```
