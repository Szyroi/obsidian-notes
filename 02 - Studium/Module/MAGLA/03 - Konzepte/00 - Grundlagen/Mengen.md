# Mengen

## Definition
Eine Menge ist eine **ungeordnete Sammlung von Elementen**, z. B. A = {1,2,3}.

## Operationen
- Vereinigung: A ∪ B = {x | x ∈ A oder x ∈ B}  
- Schnitt: A ∩ B = {x | x ∈ A und x ∈ B}  
- Differenz: A \ B = {x | x ∈ A und x ∉ B}  
- Komplement: ¬A

## Beispiel
A = {1,2,3}, B = {2,3,4}  
- A ∪ B = {1,2,3,4}  
- A ∩ B = {2,3}  
- A \ B = {1}

## Typische Fehler
- Elemente doppelt zählen  
- Reihenfolge der Elemente ist egal
