# Abbildungen und Funktionen

## 0.1 Definition
Eine **Funktion** f: A → B ordnet jedem a ∈ A genau ein b ∈ B zu.  

- **A:** Definitionsmenge  
- **B:** Zielmenge  

---

## 0.2 Eigenschaften von Funktionen
- **Injektiv (1:1):** Verschiedene Elemente in A haben verschiedene Bilder in B  
- **Surjektiv (onto):** Jedes Element in B wird getroffen  
- **Bijektiv:** injektiv + surjektiv → Umkehrfunktion möglich  

---

## 0.3 Beispiele
- f: ℝ → ℝ, f(x) = 2x → bijektiv  
- f: ℕ → ℕ, f(n) = n² → injektiv, aber nicht surjektiv  

---

## 0.4 Typische Fehler
- Injektivität / Surjektivität verwechseln  
- Definitions- und Zielmenge nicht beachten  

---

## 0.5 Klausurhinweis
- Funktionen graphisch darstellen  
- Eigenschaften prüfen  
- Umkehrfunktionen bestimmen
