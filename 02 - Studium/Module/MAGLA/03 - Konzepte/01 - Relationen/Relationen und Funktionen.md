# Relationen und Funktionen

## Definition – Relation
Eine **Relation** R zwischen Mengen A und B ist eine Teilmenge des kartesischen Produkts A × B:
$$
R \subseteq A \times B
$$

## Definition – Funktion
Eine **Funktion** f: A → B ist eine Relation, bei der jedem a ∈ A genau ein b ∈ B zugeordnet ist.

## Intuition
- Relation = beliebige Verbindung zwischen zwei Mengen  
- Funktion = „jeder Input hat genau einen Output“

## Beispiele
- Relation: „x ist größer als y“ auf ℤ: R = {(x,y) | x > y}  
- Funktion: f(x) = x² auf ℝ  

## Eigenschaften von Funktionen
- injektiv (jedes Bild höchstens einmal)  
- surjektiv (jedes Ziel wird getroffen)  
- bijektiv (injektiv + surjektiv)

## Typische Fehler
- Jede Relation ist keine Funktion!  
- Vertauschen von Definitions- und Zielmenge

## Klausurhinweis
- Beispiele als Mengen schreiben  
- Injektivität, Surjektivität prüfen
