### HTTP(s)
*AKA Hyper Text Transfer Protocol (secure)*
HTTP übermittelt normalerweise HTML
HTTP ist zustandslos
Request-Response format
HTTP hat Methoden GET/POST/...
Response codes 200,404,500,...
Request/Response Header

HTML kann andere dinge einbinden (js, css, medien, ...)

Browser Cached lokal antworten (je nach header)

Sessions werden klassisch via Cookies verwaltet
- Kleine Textmengen die vom Server beim Client gespeichert werden, die der Client dann immer wieder mitsendet
- Können als persistenter Identifier für Tracking verwendet werden
