# Data Link Layer

## Erklärung
- a
- Vorlesung: [[Vorlesung-03.pdf]]

Lokale Zustellung von Rahmen (engl. Frames) zwischen Nodes im gleichen Netzwerk. Knoten (hier Switche) leiten Pakete zwischen Nodes weiter. Gewährleistung einer weitestgehend fehlerfreien Übertragung auf dem physikalischen Medium (häufig). Zugriff auf das Übertragungsmedium regeln. Die Adressierung ist eine MAC-Adresse der Netzwerkkarte.
